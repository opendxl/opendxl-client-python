Prior to running any of the examples, you must complete the following:

  • Install the Python SDK (Python SDK Installation)

  • Provision the client to use with the samples (Provisioning Overview)

    The dxlclient.config file located in the sample sub-directory of the Python DXL SDK must be populated.

    NOTE: If Command Line Interface (CLI) provisioning is being performed, specify the sample directory as the output location for the provisioning operation.
